# 漏洞扫描系统 python476

## 项目概述

本项目是一款基于Python语言的网络安全管理系统，旨在为用户提供全方位的漏洞扫描与服务器资源监控服务。系统结合了轻量级Web框架Flask和MongoDB数据库，以及专业的漏扫工具OpenVAS和AWVS，实现了对服务器和Web漏洞的有效检测。

## 技术栈

- **开发语言：** Python3
- **Web框架：** Flask
- **数据库：** MongoDB
- **漏扫工具：** OpenVAS、AWVS

## 功能模块

- 服务器资源监控
- 端口扫描
- 服务器漏洞扫描
- Web漏洞扫描
- 域名解析
- 敏感目录查找
- 中间件漏洞扫描
- 日志爬取
- 邮件告警

## 系统角色

- 管理员：负责系统配置、任务调度、报告导出等操作。

## 运行环境

- 推荐操作系统：Linux、Windows等支持Python3的环境
- Python版本：Python3
- 第三方依赖库：Flask、MongoDB、OpenVAS、AWVS等

## 部署步骤

1. 环境准备：安装Python3、MongoDB、Flask及其依赖库。
2. 配置漏扫工具：正确配置OpenVAS和AWVS。
3. 部署应用：根据目录结构部署项目。
4. 启动服务：运行项目启动脚本，启动Web服务。

## 目录结构

```plaintext
/
├── app/ # Flask应用目录
│ ├── views/ # 视图目录
│ ├── templates/ # 模板目录
│ ├── static/ # 静态文件目录
│ └── models/ # 数据模型目录
├── scans/ # 扫描任务目录
├── reports/ # 报告输出目录
└── scripts/ # 脚本目录
```
